## 🚀 Schnellstart (unter 1 Minute)

1. **Download**
   - Klick auf „Code“ → „Download ZIP“

2. **Öffne typst.app**
   👉 https://typst.app

3. **Neues Projekt erstellen**

4. **Dateien per Drag & Drop hochladen**

5. **Beispiel öffnen**
   - `examples/EA1_A1.typ`

6. **PDF exportieren** ✅
